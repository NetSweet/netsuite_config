## Installation

Add this line to your application's Gemfile:

```ruby
gem install netsuite_config
```

Make sure you have these ENV in place

```bash
NETSUITE_EMAIL=
NETSUITE_PASSWORD=
NETSUITE_ACCOUNT=
NETSUITE_ROLE=
NETSUITE_API=
NETSUITE_PRODUCTION=
```

## Usage

```bash
netsuite_config push
netsuite_config pull
netsuite_config diff
```
